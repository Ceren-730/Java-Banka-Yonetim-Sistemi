🏦 Java Banka Yönetim Sistemi (Bank Management System)
Bu proje, Java programlama dilinin Nesne Yönelimli Programlama (OOP) prensipleri kullanılarak geliştirilmiş, konsol tabanlı bir banka hesabı yönetim simülasyonudur.

🚀 Temel Özellikler
Hesap Oluşturma: Kullanıcıdan alınan bilgilerle benzersiz hesap tanımlama.

Bakiye Yönetimi: Hesap numarası üzerinden mevcut hesaba güvenli para yatırma işlemi.

Dinamik Listeleme: ArrayList kullanarak tüm kayıtlı müşterileri ve güncel bakiyelerini görüntüleme.

Hata Kontrolü: Geçersiz bakiye girişi veya olmayan hesap numarası sorgularında kullanıcıyı bilgilendirme.

🛠 Teknik Detaylar
Sınıf Yapısı (Encapsulation): HesapKarti sınıfı ile veriler korunmuş (private), getter/setter ve metodlar ile yönetilmiştir.

Koleksiyonlar: Veri depolama için ArrayList<HesapKarti> yapısı kullanılmıştır.

Algoritma: Para yatırma işleminde liste içinde döngü ile arama (find) ve kontrol bayrağı (flag) mantığı uygulanmıştır.

📖 Nasıl Çalıştırılır?
Bilgisayarınızda Java JDK'nın yüklü olduğundan emin olun.

Projeyi bir IDE (NetBeans, IntelliJ, Eclipse vb.) içine aktarın.

Main.java dosyasını sağ tıklayıp Run diyerek başlatın.

Konsoldaki menüyü (1, 2, 3, 0) kullanarak işlemleri gerçekleştirin.
